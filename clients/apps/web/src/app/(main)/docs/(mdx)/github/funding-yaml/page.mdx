## Adding Polar to FUNDING.yaml

You can review the [GitHub
documentation](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/displaying-a-sponsor-button-in-your-repository)
or...

1. Go to the desired repository on GitHub
2. Create or modify `.github/FUNDING.yaml`
3. Add the configuration below

```yaml
polar: YOUR_POLAR_ORGANIZATION_SLUG
```

4. Commit and you're done
5. (Optional) Add some beautiful [README embeds](/docs/github/embeds) while you're at it.
