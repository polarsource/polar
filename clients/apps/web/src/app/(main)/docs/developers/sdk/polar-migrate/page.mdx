# polar-migrate

Ready to make the jump from Lemon Squeezy to Polar? Use the `polar-migrate` CLI
tool to quickly and easily migrate your existing Lemon Squeezy products to
Polar.

<video src="/assets/docs/polar-migrate.mp4" muted autoPlay loop="1" />

## Getting Started

```bash
npx polar-migrate
```

## Supported Migrations

- Products & Variants
- License Keys
- Associated Files
- Discount Codes
