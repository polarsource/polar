# Tools

Polar offers a wide array of command line tools & utilities which can help you to get up & running in no-time.

## polar-init

The fastest way to get started with Polar in your own app. Setup SaaS products & the necessary files for your framework in seconds.

```bash
npx polar-init
```

#### Supported Frameworks

- Next.js
- Nuxt.js

[Learn more](/docs/tools/polar-init)
