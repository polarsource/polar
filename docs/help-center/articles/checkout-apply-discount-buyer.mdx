---
title: "How do I apply a discount or promo code at checkout?"
sidebarTitle: "Apply a discount code"
description: "Enter a discount or promotion code on the checkout page as a customer."
audience: buyer
---

<Info>
**Audience:** Buyer — end customers completing a purchase at checkout or in the customer portal.
</Info>

On the checkout page, find the **Discount** or **Promo code** field if the seller enabled it. Enter the code and apply it before you pay. The total should update to reflect the discount.

If the field is missing, the seller may not allow codes on that link, or the product may not be eligible. If a code is rejected, check spelling and expiry, or ask the seller for a valid code.
