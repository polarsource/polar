<Info>
**Audience:** Merchant — Polar sellers using the dashboard or APIs.
</Info>

In the Polar dashboard, create a **checkout link** for a product or price. Copy the generated URL and send it to customers by email, chat, or your website.

For programmatic use, you can also create checkout sessions via the API—see [Checkout Links](/features/checkout/links) and the [Checkout API](/api-reference/checkouts/create-session).
