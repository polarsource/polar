<Info>
**Audience:** Merchant — Polar sellers using the dashboard or APIs.
</Info>

Use Polar’s **sandbox** environment for development and staging. Create products and checkout links or sessions in sandbox, then complete payment with [Stripe test card numbers](https://docs.stripe.com/testing#cards) (for example `4242 4242 4242 4242` with any future expiry and CVC).

See [Sandbox](/integrate/sandbox) for how sandbox differs from production and how to switch environments in your integration.
