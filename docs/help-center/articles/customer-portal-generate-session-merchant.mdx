<Info>
**Audience:** Merchant — Polar sellers using the dashboard or APIs.
</Info>

Use the **Customer Portal API** to [create a session](/api-reference/customer-portal/sessions/create) for a customer. Return the session URL to your app and redirect or link the customer to it. Sessions expire; generate a new one when the customer needs access again.

See [Customer portal](/features/customer-portal) for hosted portal behavior and how sessions fit into your integration.
