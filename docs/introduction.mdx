## Quickstart from CLI

The fastest way to get started with Polar in your Next.js or Nuxt.js app. Setup SaaS products & the necessary files for your framework in seconds.

```bash
npx polar-init
```

**Supported Frameworks**
- Next.js
- Nuxt.js

**Open Source**

The code is open source and available on [GitHub](https://github.com/polarsource/polar-init)
