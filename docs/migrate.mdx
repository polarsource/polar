## Lemon Squeezy

Ready to make the jump from Lemon Squeezy to Polar? Use the `polar-migrate` CLI tool to quickly and easily migrate your existing Lemon Squeezy products to Polar.

### Getting Started

```bash Terminal
npx polar-migrate@latest
```

### Supported Migrations

* Products & Variants
* License Keys
* Associated Files
* Discount Codes
* Customers

This tool is not able to move **active** subscriptions from your Lemon Squeezy store.

### Open Source

The code for the CLI is open source and available on GitHub

[View Code on GitHub](https://github.com/polarsource/polar-migrate)

## Paddle

There's no automated tool for Paddle yet. Recreate your products and prices in Polar, then move customers over at their next renewal.

If you use Paddle's subscription **pause** to freeze accounts, for example seasonal plans that stop billing during the off-season, Polar has a direct equivalent. Pausing stops billing at the end of the current period and revokes benefits. The subscription resumes later, on a date you set or whenever you choose, and keeps its payment method on file so there's no second checkout. See [Pause and resume](/features/subscriptions/manage#pause-and-resume) for the API and behavior.
